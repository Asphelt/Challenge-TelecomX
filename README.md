# Challenge-TelecomX
Este proyecto analiza la evasión de clientes (churn) de una empresa de telecomunicaciones llamada Telecom X. El objetivo es entender por qué algunos clientes cancelan el servicio y qué características están más relacionadas con esa decisión.

El análisis se realiza usando Python. Los datos provienen de un archivo en formato JSON que contiene información de los clientes, los servicios contratados, la facturación y el estado de churn.

El proyecto incluye un notebook principal donde se cargan los datos, se limpian y transforman, se crean nuevas variables y se realiza un análisis exploratorio para identificar patrones. Al final se presentan conclusiones y algunas recomendaciones generales.

Para ejecutar el proyecto se necesita tener Python 3.8 o superior y un entorno como Jupyter Notebook o Google Colab. También se utilizan las librerías pandas, numpy, matplotlib y seaborn.

Los archivos principales del proyecto son el notebook de análisis, el archivo JSON con los datos y un archivo con el diccionario de datos que explica el significado de cada columna.

Para ejecutar el análisis, basta con abrir el notebook, asegurarse de que el archivo JSON esté en la misma carpeta y ejecutar todas las celdas en orden. El notebook está preparado para ejecutarse desde cero sin errores si se sigue ese orden.

Durante el análisis se observa que el churn es más común en contratos mensuales, en clientes con poco tiempo de permanencia y en algunos casos en clientes con cargos mensuales más altos. Estos resultados ayudan a entender mejor el comportamiento de los clientes y pueden servir como base para estrategias de retención.

Este proyecto tiene un enfoque educativo y práctico, orientado a aprender y aplicar conceptos básicos de análisis de datos y análisis exploratorio.
